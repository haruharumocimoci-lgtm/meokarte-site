# HPミセガオ リデザイン - デザインブレインストーム

## 3つのスタイリスティックアプローチ

### 1. 「Warm Authority」 - 温かみのある権威性
- **Very Brief Intro**: 法律事務所やコンサルティングファームのような重厚感と、飲食店に寄り添う温かみを融合。深いネイビーとゴールドアクセントで「プロフェッショナルだけど親しみやすい」を実現。
- **Probability**: 0.07

### 2. 「Craft Studio」 - 職人工房
- **Very Brief Intro**: 手仕事の温もりと現代的なクリーンさを両立。和紙のようなテクスチャ、墨のようなタイポグラフィ、余白を活かした日本的な美意識で「丁寧に作られている」という信頼感を演出。
- **Probability**: 0.04

### 3. 「Elevated SaaS」 - 洗練されたSaaS
- **Very Brief Intro**: Stripe・Notionのようなモダンテック企業の洗練さを飲食店向けサービスに適用。微細なグラデーション、ガラスモーフィズム、精密なグリッドで「テクノロジーで支える安心感」を表現。
- **Probability**: 0.06

## 選択: 「Craft Studio」 - 職人工房

### Design Movement
**和モダニズム（Wa-Modernism）** — 日本の伝統的な美意識（間・余白・素材感）と、スイスタイポグラフィの精密さを融合させたアプローチ。無印良品やAesopのような「静かだが確かな品質」を感じさせるデザイン。

### Core Principles
1. **Ma（間）の活用** — 余白を恐れず、情報に呼吸を与える。詰め込まない勇気が信頼感を生む。
2. **素材感（Texture as Trust）** — 和紙、木目、墨のにじみなど、手触りを感じさせるテクスチャが「人の手で丁寧に作られている」ことを伝える。
3. **控えめな力強さ** — 派手さではなく、タイポグラフィの重みと構図の安定感で権威性を示す。
4. **一点の色気** — モノトーンの中に差し込む一色（深い朱色）が、記憶に残るアクセントとなる。

### Color Philosophy
- **Primary**: 墨色（Sumi） `#1a1a2e` — 黒ではなく、深みのある墨。知性と落ち着き。
- **Secondary**: 生成り（Kinari） `#f5f0e8` — 白ではなく、和紙のような温かみのあるオフホワイト。
- **Accent**: 朱（Shu） `#c23616` — 判子や暖簾を想起させる、日本的な赤。信頼の証。
- **Muted**: 灰青（Haiao） `#6b7b8d` — 説明文や補助テキストに使う、落ち着いた灰青。
- **Surface**: 薄墨（Usuzumi） `#2d2d44` — カードやセクション背景に使う、柔らかい暗色。

### Layout Paradigm
**縦書きインスピレーションの非対称レイアウト** — 日本の縦書き文化から着想を得た、左右非対称のグリッド。大きな見出しは画面の左側に寄せ、説明文は右側に流す。セクション間は大きな余白で区切り、スクロールの「間」を意識する。ただし実際のテキストは横書き。

### Signature Elements
1. **墨のストローク** — セクション区切りや装飾に、筆で引いたようなストロークを配置。手仕事感の象徴。
2. **判子モチーフ** — CTAボタンやバッジに、朱色の判子（スタンプ）のようなデザインを採用。「承認・信頼」の視覚的メタファー。
3. **和紙テクスチャ背景** — 背景に微細な和紙のテクスチャを敷き、デジタルの冷たさを和らげる。

### Interaction Philosophy
控えめだが確かな反応。ホバーは「墨がにじむ」ように柔らかく広がり、クリックは「判子を押す」ように確実に沈む。過剰なアニメーションは避け、動きは常に目的を持つ。

### Animation
- **スクロール連動**: セクションがビューポートに入る際、下から静かにフェードイン（200ms ease-out）
- **ホバー**: 要素が0.98にスケールダウンし、影が深くなる（150ms）
- **CTA**: 押下時に0.95スケール + 朱色の波紋エフェクト（300ms）
- **数字**: 料金表示はカウントアップアニメーション
- **ステップ**: 横からスライドイン（stagger 100ms）

### Typography System
- **Display / Heading**: Noto Serif JP (700) — 明朝体の格調高さ。見出しに使用。
- **Body**: Noto Sans JP (400/500) — 可読性の高いゴシック体。本文に使用。
- **Accent / Label**: Noto Sans JP (300) — 軽いウェイトでサブラベルやキャプションに。
- **English**: Inter (400/600) — セクションラベルの英語テキストに。
- **Hierarchy**: Display 48-64px / H2 32-40px / H3 24px / Body 16-18px / Caption 14px

### Brand Essence
**「飲食店オーナーのための、手仕事品質のデジタルパートナー」** — 技術力を持ちながらも、一軒一軒に向き合う職人気質のサービス。
- 3つの性格形容詞: **誠実・丁寧・頼もしい**

### Brand Voice
- ヘッドラインは「です・ます」ではなく「体言止め」や「〜します」で簡潔に。押し付けない、でも確信を持った語り口。
- 例1: 「お店の顔、つくります。」（既存コピーをそのまま活用）
- 例2: 「実物を見てから、決めてください。」

### Wordmark & Logo
「顔」の漢字を朱色の判子風にデザインしたシンボルマーク + 「HPミセガオ」のNoto Serif JPロゴタイプ。判子は正方形ではなく、少し角が丸い長方形で現代的に。

### Signature Brand Color
**朱（Shu）** `#c23616` — 判子・暖簾・鳥居を想起させる、日本の「信頼と承認」の色。


## Style Decisions

- 主要セクションは料金・FAQを除き、見出しと本文/カードを左右にずらした非対称編集レイアウトを基本とし、中央揃えのSaaS型ブロックを連続させない。
- 朱 `#c23616` はCTA、判子、重要な価格・数字、信頼を示す小ラベルに限定し、装飾色ではなく「承認の印」として使う。
- カードやセクション背景には必ず和紙・墨ムラ・細い罫線・判子のいずれかの素材的サインを含め、フラットな無地カードだけで完結させない。
